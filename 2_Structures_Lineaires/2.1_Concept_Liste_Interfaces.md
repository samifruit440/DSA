# 2.1 Concept de Liste et Interfaces Standards

## Définition de la liste

Une **liste** est une structure de données abstraite qui permet de stocker une collection ordonnée d'éléments et d'accéder librement à n'importe quel élément.

Caractéristiques :
- Collection linéaire et ordonnée
- Accès direct à n'importe quel élément
- Taille variable

## Méthodes de base associées

```java
// Interface List<E>
void add(E element);           // Ajouter un élément à la fin
void add(int index, E element); // Ajouter à une position
E remove(int index);            // Supprimer à une position
E get(int index);               // Récupérer un élément
void set(int index, E element); // Remplacer un élément
int size();                      // Obtenir la taille
boolean isEmpty();              // Vérifier si vide
```

## Interface Collection et Iterable

### Interface Collection<E>

Mère de toutes les collections. Définit les opérations communes :

```java
public interface Collection<E> extends Iterable<E> {
    boolean add(E e);
    boolean remove(Object o);
    boolean contains(Object o);
    int size();
    boolean isEmpty();
    void clear();
    Iterator<E> iterator();
}
```

### Interface Iterable<E>

Permet d'itérer sur les éléments :

```java
public interface Iterable<E> {
    Iterator<E> iterator();
}
```

## Interface Iterator (hasNext, next, remove) et complexité

L'**Iterator** fournit une manière sécurisée de parcourir une collection.

```java
public interface Iterator<E> {
    boolean hasNext();      // O(1) - Vérifier s'il y a un élément suivant
    E next();               // O(1) - Obtenir l'élément suivant
    void remove();          // O(n) - Supprimer l'élément courant
}
```

### Utilisation

```java
List<String> liste = new ArrayList<>();
Iterator<String> iter = liste.iterator();

while (iter.hasNext()) {
    String element = iter.next();  // O(1)
    System.out.println(element);
}

// Ou avec for-each (utilise Iterator en interne)
for (String element : liste) {
    System.out.println(element);
}
```

## Interface List<E>

Extension de Collection avec accès indexé :

```java
public interface List<E> extends Collection<E> {
    E get(int index);
    E set(int index, E element);
    void add(int index, E element);
    E remove(int index);
    int indexOf(Object o);
    int lastIndexOf(Object o);
    ListIterator<E> listIterator();
    ListIterator<E> listIterator(int index);
    List<E> subList(int fromIndex, int toIndex);
}
```

## Interface ListIterator

Sous-interface d'Iterator permettant un parcours bidirectionnel :

```java
public interface ListIterator<E> extends Iterator<E> {
    // Méthodes d'Iterator
    boolean hasNext();
    E next();
    void remove();
    
    // Méthodes supplémentaires
    boolean hasPrevious();      // Vérifier s'il y a un élément précédent
    E previous();               // Obtenir l'élément précédent
    int nextIndex();            // Index du prochain élément
    int previousIndex();        // Index du précédent élément
    void set(E e);              // Remplacer l'élément courant
    void add(E e);              // Ajouter un élément à la position courante
}
```

### Exemple d'utilisation

```java
List<Integer> liste = new ArrayList<>();
liste.addAll(Arrays.asList(1, 2, 3, 4, 5));

ListIterator<Integer> iter = liste.listIterator();

// Parcours avant
while (iter.hasNext()) {
    int val = iter.next();
    if (val == 3) iter.set(30); // Remplacer 3 par 30
}

// Parcours arrière
while (iter.hasPrevious()) {
    int val = iter.previous();
    System.out.println(val);
}
```
