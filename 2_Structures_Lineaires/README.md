- **Listes** : Collections ordonnées permettant l'accès libre
- **Piles** : Structure LIFO (Last In, First Out)
- **Files** : Structure FIFO (First In, First Out)

- Interfaces Collection et Iterable
- Implémentations (ArrayList, LinkedList)
