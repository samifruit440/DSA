# 10. Structures d'Ensembles Disjoints

## Vue d'ensemble

Structure de données permettant de gérer des **ensembles disjoints** (non chevauchants) et de fusionner progressivement ces ensembles.

Applications :
- Détection de cycles avec Kruskal
- Génération de labyrinthes
- Composantes connexes
- Problèmes d'équivalence
