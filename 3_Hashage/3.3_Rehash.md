# 3.3 Rehash

## Nécessité de redimensionner la table

À mesure que la table se remplit, le **facteur de charge** augmente, ce qui augmente les collisions et dégrade les performances.

$$fc = \frac{\text{nombre d'éléments}}{\text{taille de la table}}$$

**Seuils typiques** :
- **Chaînage** : fc = 1.0 (on peut tolérer une moyenne d'un élément par bucket)
- **Sondage quadratique** : fc ≤ 0.5 (maximum 50% de remplissage)
- **Double hashing** : fc ≤ 0.7-0.8

Quand le facteur de charge dépasse le seuil, la table doit être **redimensionnée**.

## Processus de Rehash

Le rehash suit ces étapes :

### 1. Créer une nouvelle table plus grande

```
Ancienne table : taille = 11
Nouvelle table : taille = 23 (prochain nombre premier > 2*11)
```

**Important** : La nouvelle taille doit être un **nombre premier** pour maintenir une bonne distribution.

```java
private static int nextPrime(int n) {
    while (!isPrime(n)) n++;
    return n;
}

private static boolean isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```

### 2. Recalculer les hash pour tous les éléments

Avec une nouvelle taille de table, les indices calculés par la fonction hash changent.

```
Ancienne table (taille = 11) :
H(27) = 27 % 11 = 5

Nouvelle table (taille = 23) :
H(27) = 27 % 23 = 4  ← Index différent !
```

### 3. Recopier les éléments

Insérer tous les anciens éléments dans la nouvelle table à leurs nouveaux indices.

```java
public void rehash() {
    Entry<K, V>[] oldTable = table;
    
    // Créer une nouvelle table plus grande
    int newSize = nextPrime(table.length * 2);
    table = new Entry[newSize];
    size = 0;
    
    // Recopier tous les éléments
    for (Entry<K, V> entry : oldTable) {
        if (entry != null) {
            put(entry.key, entry.value);
        }
    }
}
```

## Implémentation complète avec Rehash

```java
public class HashTableWithRehash<K, V> {
    private static final int INITIAL_SIZE = 11;
    private static final double LOAD_FACTOR_THRESHOLD = 0.75;
    
    private class Entry<K, V> {
        K key;
        V value;
        Entry(K key, V value) {
            this.key = key;
            this.value = value;
        }
    }
    
    private Entry<K, V>[] table;
    private int size;
    
    @SuppressWarnings("unchecked")
    public HashTableWithRehash() {
        table = new Entry[INITIAL_SIZE];
        size = 0;
    }
    
    private int hash(K key) {
        return Math.abs(key.hashCode()) % table.length;
    }
    
    public void put(K key, V value) {
        // Vérifier si rehash nécessaire
        if ((double) (size + 1) / table.length > LOAD_FACTOR_THRESHOLD) {
            rehash();
        }
        
        int index = hash(key);
        
        // Sondage linéaire pour trouver une position libre
        while (table[index] != null) {
            if (table[index].key.equals(key)) {
                // Clé existe déjà
                table[index].value = value;
                return;
            }
            index = (index + 1) % table.length;
        }
        
        table[index] = new Entry<>(key, value);
        size++;
    }
    
    public V get(K key) {
        int index = hash(key);
        
        while (table[index] != null) {
            if (table[index].key.equals(key)) {
                return table[index].value;
            }
            index = (index + 1) % table.length;
        }
        
        return null;
    }
    
    @SuppressWarnings("unchecked")
    private void rehash() {
        Entry<K, V>[] oldTable = table;
        int oldSize = size;
        
        // Créer nouvelle table
        int newSize = nextPrime(table.length * 2);
        table = new Entry[newSize];
        size = 0;
        
        // Recopier tous les éléments
        for (Entry<K, V> entry : oldTable) {
            if (entry != null) {
                put(entry.key, entry.value);
            }
        }
        
        System.out.println("Rehash : " + oldTable.length + " → " + newSize);
    }
    
    private static int nextPrime(int n) {
        while (!isPrime(n)) n++;
        return n;
    }
    
    private static boolean isPrime(int n) {
        if (n < 2) return false;
        if (n == 2) return true;
        if (n % 2 == 0) return false;
        for (int i = 3; i * i <= n; i += 2) {
            if (n % i == 0) return false;
        }
        return true;
    }
    
    public int size() {
        return size;
    }
    
    public int getTableSize() {
        return table.length;
    }
}
```

## Exemple d'utilisation

```java
HashTableWithRehash<String, Integer> map = new HashTableWithRehash<>();

// Insérer des éléments
for (int i = 1; i <= 10; i++) {
    map.put("clé" + i, i * 100);
    System.out.println("Ajouté : taille=" + map.size() + 
                       ", table size=" + map.getTableSize());
    // Sortie :
    // Ajouté : taille=1, table size=11
    // Ajouté : taille=2, table size=11
    // ...
    // Ajouté : taille=9, table size=11
    // Rehash : 11 → 23
    // Ajouté : taille=10, table size=23
}
```

## Complexité du Rehash

### Coût amorti

Bien que le rehash soit O(n), il se fait rarement, donc le coût **amorti** est O(1).

| Opération | Coût | Coût amorti |
|-----------|------|-----------|
| `put()` sans rehash | O(1) | O(1) |
| `put()` avec rehash | O(n) | O(1) amorti |

### Analyse

- Rehash après avoir inséré n éléments
- Coût total pour n insertions : O(n) + O(n/2) + O(n/4) + ... = O(2n) = O(n)
- **Coût amorti par insertion** : O(n) / n = O(1)

## Optimisations

### 1. Lazy deletion

Pour le sondage, marquer les éléments supprimés comme "effacés" sans réellement les retirer, pour éviter les ruptures dans les séquences de sondage.

### 2. Robin Hood Hashing

Amélioration du sondage qui réduit la variance des distances de sondage, donnant une distribution plus uniforme.

### 3. Croissance adaptive

Augmenter progressivement au lieu de doubler directement.
