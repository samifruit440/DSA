# 3. Hashage et Tables de Dispersement

Cette section couvre les techniques de hachage pour implémenter des structures de données efficaces de recherche rapide.

## Concepts clés

- **Fonction de dispersement** : Transforme une clé en index
- **Collisions** : Quand deux clés différentes produisent le même hash
- **Résolution de collisions** : Chaînage ou sondage
- **Rehashing** : Redimensionnement de la table
