# 9. Recherche de Patron et Programmation Dynamique

## Vue d'ensemble

Cette section couvre deux techniques majeures de résolution de problèmes :

- **Recherche de Patron** : Trouver un motif (pattern) dans un texte de manière efficace
- **Programmation Dynamique** : Décomposer un problème complexe en sous-problèmes récursifs avec memoïsation
