# 5.2 Parcours d'Arbres

## Parcours basés sur la récursivité

### Pré-ordre (Preorder) : Nœud, Gauche, Droite

Visiter le **nœud actuel** avant ses enfants.

```
Arbre :
    1
   / \
  2   3
 / \
4   5

Parcours : 1, 2, 4, 5, 3
```

```java
public void preorder(TreeNode node) {
    if (node == null) return;
    System.out.println(node.value);  // Visiter d'abord
    preorder(node.left);
    preorder(node.right);
}
```

**Utilité :** Copier l'arbre, exprimer les opérations avant les opérandes

### In-ordre (Inorder) : Gauche, Nœud, Droite

Visiter le **nœud actuel** entre ses deux enfants.

```
Arbre :
    1
   / \
  2   3
 / \
4   5

Parcours : 4, 2, 5, 1, 3
```

```java
public void inorder(TreeNode node) {
    if (node == null) return;
    inorder(node.left);
    System.out.println(node.value);  // Visiter au milieu
    inorder(node.right);
}
```

**Utilité :** **CRUCIAL pour ABR** - produit les éléments en ordre croissant

### Post-ordre (Postorder) : Gauche, Droite, Nœud

Visiter le **nœud actuel** après ses enfants.

```
Arbre :
    1
   / \
  2   3
 / \
4   5

Parcours : 4, 5, 2, 3, 1
```

```java
public void postorder(TreeNode node) {
    if (node == null) return;
    postorder(node.left);
    postorder(node.right);
    System.out.println(node.value);  // Visiter en dernier
}
```

**Utilité :** Supprimer l'arbre, évaluer expressions mathématiques

## Parcours par niveau (Level-order)

Visiter les nœuds **niveau par niveau** de gauche à droite.

```
Arbre :
    1
   / \
  2   3
 / \   \
4   5   6

Parcours : 1, 2, 3, 4, 5, 6
```

### Implémentation non-récursive avec Queue

```java
public void levelOrder(TreeNode root) {
    if (root == null) return;
    
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    
    while (!queue.isEmpty()) {
        TreeNode node = queue.poll();
        System.out.println(node.value);
        
        if (node.left != null) queue.offer(node.left);
        if (node.right != null) queue.offer(node.right);
    }
}
```

**Complexité :** O(n) - visite chaque nœud une fois

## Résumé des parcours

| Parcours | Ordre | Utilité | Récursif |
|----------|-------|---------|----------|
| Pré-ordre | Nœud, Gauche, Droite | Copie, expressions préfixes | ✓ |
| In-ordre | Gauche, Nœud, Droite | ABR → ordre croissant | ✓ |
| Post-ordre | Gauche, Droite, Nœud | Suppression, expressions postfixes | ✓ |
| Par niveau | Niveau par niveau | BFS, largeur d'abord | File |
